Loop in JavaScript

In JavaScript, loops allow you to repeat a block of code multiple times. There are several types of loops to suit different needs. Here's a breakdown:

  1. for Loop

    • The for loop is typically used when you know beforehand how many times you want to iterate through a block of code.

      • Syntax:

                                            for (initialization; condition; increment) {
                                    // code to be executed
                                }

      • Example:

                                            for (let i = 0; i < 5; i++) {
                                    console.log(i);
                                }
                                // Output: 0, 1, 2, 3, 4

  2. while Loop

    • The while loop is used when you want to repeat a block of code while a condition is true.

      • Syntax:

                                            while (condition) {
                                        // code to be executed
                                    }

      • Example:

                                            let i = 0;
                                    while (i < 5) {
                                        console.log(i);
                                        i++;
                                    }
                                    // Output: 0, 1, 2, 3, 4

  3. do...while Loop

    • The do...while loop is similar to the while loop, but it guarantees that the block of code will run at least once, even if the condition is false.

      • Syntax:

                                            do {
                                        // code to be executed
                                    } while (condition);

      • Example:

                                            let i = 0;
                                    do {
                                        console.log(i);
                                        i++;
                                    } while (i < 5);
                                    // Output: 0, 1, 2, 3, 4

  4. for...in Loop

    • The for...in loop is used to iterate over the keys (properties) of an object or the indices of an array.

      • Syntax:

                                            for (let key in object) {
                                        // code to be executed
                                    }

      • Example:

                                            let i = 0;
                                    do {
                                        console.log(i);
                                        i++;
                                    } while (i < 5);
                                    // Output: 0, 1, 2, 3, 4

      • Example (for an array):

                                                let arr = [10, 20, 30];
                                        for (let value of arr) {
                                            console.log(value);
                                        }
                                        // Output: 10, 20, 30

      • Example(for an string):

                                                    let str = "Hello";
                                            for (let char of str) {
                                                console.log(char);
                                            }
                                            // Output: H, e, l, l, o

Loop Control Statements

  1. break

    • Used to exit the loop immediately, even if the loop condition hasn't been met.

    • Example:

                                            for (let i = 0; i < 10; i++) {
                                    if (i === 5) {
                                        break; // Exit the loop when i is 5
                                    }
                                    console.log(i);
                                }
                                // Output: 0, 1, 2, 3, 4

  2. continue

    • Used to skip the current iteration and continue with the next iteration of the loop.

    • Example:

                                      for (let i = 0; i < 5; i++) {
                                if (i === 3) {
                                    continue; // Skip the rest of the loop when i is 3
                                }
                                console.log(i);
                            }
                            // Output: 0, 1, 2, 4
