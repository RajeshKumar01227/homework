Sting in JavaScript

A string in JavaScript is a sequence of characters used to represent text. Strings are enclosed in either single quotes (''), double quotes (""), or backticks (``).

  1. Creating Strings

    • You can create a string using:

      • Syntax:

                                            let str1 = "Hello, World!";  // Double quotes
                                    let str2 = 'JavaScript';     // Single quotes
                                    let str3 = `Template String`; // Backticks (Template literals)

  2. String Properties

    • Strings in JavaScript are immutable (cannot be changed) but can be reassigned.

      • Length of a String

        You can get the number of characters using .length:

                                            let message = "Hello";
                                    console.log(message.length); // Output: 5

  3. String Methods

    • Changing Case

      • Syntax:

                                            let text = "JavaScript";
                                    console.log(text.toUpperCase()); // "JAVASCRIPT"
                                    console.log(text.toLowerCase()); // "javascript"

    • Extracting Parts of a String

      • .slice(start, end) → Extracts a portion of a string.
      • .substring(start, end) → Similar to slice but doesn't accept negative indexes.
      • .substr(start, length) → Extracts a part of a string based on a starting index and length.

      • Syntax:

                                            let str = "Hello, World!";
                                    console.log(str.slice(0, 5));     // "Hello"
                                    console.log(str.substring(0, 5)); // "Hello"
                                    console.log(str.substr(7, 5));    // "World"

    • Finding a Substring

      • .indexOf("text") → Returns the index of the first occurrence.
      • .lastIndexOf("text") → Returns the index of the last occurrence.
      • .includes("text") → Returns true if the string is found.
      • .startsWith("text") → Checks if the string starts with a specific substring.
      • .endsWith("text") → Checks if the string ends with a specific substring

      • Syntax:

                                            let text = "Hello, JavaScript!";
                                    console.log(text.indexOf("JavaScript"));  // 7
                                    console.log(text.lastIndexOf("o"));       // 4
                                    console.log(text.includes("Java"));       // true
                                    console.log(text.startsWith("Hello"));    // true
                                    console.log(text.endsWith("!"));          // true

    • Replacing Text

      • .replace("old", "new") → Replaces the first occurrence.
      • .replaceAll("old", "new") → Replaces all occurrences.

      • Syntax:

                                                let sentence = "I love JavaScript!";
                                        console.log(sentence.replace("JavaScript", "Python")); // "I love Python!"
                                        console.log(sentence.replaceAll("o", "O")); // "I lOve JavaScript!"

    • Trimming White Spaces

      • .trim() → Removes spaces from both ends.
      • .trimStart() → Removes spaces from the start.
      • .trimEnd() → Removes spaces from the end.

        • Syntax:

                                                    let msg = "   Hello, World!   ";
                                            console.log(msg.trim());       // "Hello, World!"
                                            console.log(msg.trimStart());  // "Hello, World!   "
                                            console.log(msg.trimEnd());    // "   Hello, World!"

  4. String Concatenation

    • You can combine strings using + or concat():

      • Syntax:

                                            let firstName = "John";
                                    let lastName = "Doe";
                                    console.log(firstName + " " + lastName); // "John Doe"
                                    console.log(firstName.concat(" ", lastName)); // "John Doe"

    • Using Template Literals (``)

      • Syntax:

                                            let name = "Alice";
                                    let age = 25;
                                    console.log(`My name is ${name} and I am ${age} years old.`);
                                    // "My name is Alice and I am 25 years old."

  5. Splitting a String

    • .split(separator, limit) converts a string into an array.

      • Syntax:

                                            let sentence = "apple,banana,orange";
                                    console.log(sentence.split(",")); // ["apple", "banana", "orange"]
                                    console.log(sentence.split("", 5)); // ["a", "p", "p", "l", "e"]

  6. String Iteration

    • Using for...of:

      • Syntax:

                                            let text = "Hello";
                                    for (let char of text) {
                                   console.log(char);
                                    }
                                   // H
                                   // e
                                   // l
                                  // l
                                 // o

  7. Escape Characters

    • Use \ to include special characters:

      • Syntax:

                                            console.log("He said, \"Hello!\"");  // "He said, "Hello!""
                                    console.log('It\'s a sunny day.');  // "It's a sunny day."
                                    console.log("First Line\nSecond Line"); // New line
                                    console.log("Tab\tSpace"); // Tab space

  8. Checking if a String is Empty

      • Syntax:

                                            let str = "";
                                    console.log(str.length === 0); // true
                                    console.log(!str); // true (Falsy value)
