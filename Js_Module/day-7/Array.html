Arrays in JavaScript

A string in JavaScript is a sequence of characters used to represent text. Strings are enclosed in either single quotes (''), double quotes (""), or backticks (``).

  1. Creating Arrays

    • Using Array Literals (Recommended):

      • Syntax:

                                            let fruits = ["Apple", "Banana", "Cherry"];

    • Using Array Literals (Recommended):

      • Syntax:

                                        let colors = new Array("Red", "Green", "Blue");

    • Empty Array:

      • Syntax:

                                    let emptyArray = [];

  2. Accessing Array Elements

    • You can access elements using zero-based indexing (0 is the first element).

      • Syntax:

        You can get the number of characters using .length:

                                            let numbers = [10, 20, 30, 40];
                                    console.log(numbers[0]); // 10
                                    console.log(numbers[2]); // 30
                                    console.log(numbers[numbers.length - 1]); // Last element (40)
      • If you try to access an out-of-bounds index, you get undefined.

  3. Modifying Arrays

    • Changing an Element

      • Syntax:

                                            let fruits = ["Apple", "Banana", "Cherry"];
                                    fruits[1] = "Mango"; // Change "Banana" to "Mango"
                                    console.log(fruits); // ["Apple", "Mango", "Cherry"]

    • Adding Elements

      • Syntax:

                                            let arr = [1, 2, 3];
                                    arr[3] = 4; // Adds a new element at index 3
                                    console.log(arr); // [1, 2, 3, 4]

  4. Array Properties

    • .length Property:

    • The .length property returns the number of elements in an array.

      • Syntax:

                                            let numbers = [10, 20, 30, 40];
                                    console.log(numbers.length); // 4

  5. Array Methods

    • Adding and Removing Elements

      • Syntax:

                                            let fruits = ["Apple", "Banana", "Cherry"];

                                    // Add elements
                                    fruits.push("Mango"); // ["Apple", "Banana", "Cherry", "Mango"]
                                    fruits.unshift("Pineapple"); // ["Pineapple", "Apple", "Banana", "Cherry", "Mango"]

                                    // Remove elements
                                    fruits.pop(); // Removes "Mango"
                                    fruits.shift(); // Removes "Pineapple"
                                    console.log(fruits); // ["Apple", "Banana", "Cherry"]

    • Concatenating Arrays (.concat())

      • Syntax:

                                        let arr1 = [1, 2];
                                let arr2 = [3, 4];
                                let result = arr1.concat(arr2);
                                console.log(result); // [1, 2, 3, 4]

    • Modifying Array (.splice(start, deleteCount, item1, item2, ...))

      • Syntax:

                                    let fruits = ["Apple", "Banana", "Cherry"];
                            fruits.splice(1, 1, "Mango", "Orange"); 
                            console.log(fruits); // ["Apple", "Mango", "Orange", "Cherry"]

  6. Searching in an Array

      • Syntax:

                                            let fruits = ["Apple", "Banana", "Cherry"];
                                    console.log(fruits.indexOf("Banana")); // 1
                                    console.log(fruits.includes("Mango")); // false

  7. Sorting and Reversing

      ]

    • Changing an Element

      • Syntax:

                                            let numbers = [40, 100, 1, 5];
                                    numbers.sort((a, b) => a - b); // Ascending order
                                    console.log(numbers); // [1, 5, 40, 100]

    • Reversing (.reverse())

      • Syntax:

                                        let names = ["Alice", "Bob", "Charlie"];
                                names.reverse();
                                console.log(names); // ["Charlie", "Bob", "Alice"]

  8. Iterating Through Arrays

    • Using for Loop

      • Syntax:

                                            let arr = ["a", "b", "c"];
                                    for (let i = 0; i < arr.length; i++) {
                                        console.log(arr[i]);
                                    }

    • Using forEach()

      • Syntax:

                                        let numbers = [1, 2, 3];
                                numbers.forEach((num) => console.log(num * 2)); // 2, 4, 6

  9. Array Transformatio

    • Using .map()

      • Syntax:

                                            let numbers = [1, 2, 3, 4];
                                    let squared = numbers.map(num => num ** 2);
                                    console.log(squared); // [1, 4, 9, 16]

    • Using .filter()

      • Syntax:

                                        let numbers = [1, 2, 3, 4, 5, 6];
                                let evens = numbers.filter(num => num % 2 === 0);
                                console.log(evens); // [2, 4, 6]

    • Using .reduce()

      • Syntax:

                                    let numbers = [1, 2, 3, 4];
                            let sum = numbers.reduce((acc, num) => acc + num, 0);
                            console.log(sum); // 10

  10. Checking If an Array Contains Something

    • Checking if a Value Exists (.includes())

      • Syntax:

                                            let arr = [10, 20, 30];
                                    console.log(arr.includes(20)); // true

    • Checking If Every Element Matches (.every())

      • Syntax:

                                        let numbers = [2, 4, 6];
                                console.log(numbers.every(num => num % 2 === 0)); // true

    • Checking If Some Elements Match (.some())

      • Syntax:

                                    let numbers = [1, 3, 6];
                            console.log(numbers.some(num => num % 2 === 0)); // true (because 6 is even)
