Flex Box Properties

Flexbox in CSS

Flexbox (short for Flexible Box Layout) is a CSS layout model designed to arrange items in a one-dimensional space (either horizontally or vertically) within a container. Flexbox allows you to align items dynamically, making it easier to create complex layouts and handle responsiveness without using floats or positioning.

Key Properties of Flexbox

1. Flex Container

To activate Flexbox, you need to apply display: flex (or inline-flex) to the container element. This makes the children of this container "flex items."

Exmple

                    .container {
                        display: flex;  /* or inline-flex */
                      }

2. Flex Direction

The flex-direction property defines the direction in which flex items are laid out in the container. It controls the main axis (horizontal or vertical).

  • row (default): Items are arranged horizontally (left to right).
  • column: Items are arranged vertically (top to bottom).
  • row-reverse: Items are arranged horizontally, but in reverse order.
  • column-reverse: Items are arranged vertically, but in reverse order.

Example

                    .container {
                        flex-direction: row;
                    }

3. Flex Wrap

By default, Flexbox tries to fit all items on one line. You can use the flex-wrap property to allow the items to wrap onto the next line if there isn't enough space.

  • nowrap (default): All items will be on one line.
  • wrap: Items will wrap to the next line if needed.
  • wrap: Items will wrap to the next line if needed.
  • wrap-reverse: Items will wrap in reverse order.

Example

                    .container {
                        flex-wrap: wrap;  /* Allows wrapping */
                      }

4. Justify Content

The justify-content property aligns items along the main axis (horizontal, in a row layout, or vertical in a column layout). It helps distribute space between items.

    flex-end: Items are aligned at the end of the container.
  • flex-start: Items are aligned at the beginning of the container.
  • flex-end: Items are aligned at the end of the container.
  • center: Items are centered.
  • space-between: Items are evenly distributed, with space between them.
  • space-around: Items are evenly distributed, with space around them.
  • space-evenly: Items are evenly distributed, with equal space between them.

Example

                .container {
                    justify-content: center; /* Centers items horizontally */
                  }

5. Align items

The align-items property aligns flex items along the cross axis (vertical, in a row layout, or horizontal in a column layout).

  • flex-start: Items are aligned to the top (in a row) or left (in a column).
  • flex-end: Items are aligned to the bottom (in a row) or right (in a column).
  • center: Items are aligned at the center.
  • baseline: Items are aligned based on their baseline (typically used for text).
  • stretch (default): Items stretch to fill the container's height (in a row) or width (in a column).

Example

                .container {
                    align-items: center; /* Centers items vertically */
                  }

6. Align Content

The align-content property is used when there are multiple lines of flex items, and it aligns the lines along the cross axis (vertical or horizontal). This property works only when the flex-wrap property is set to wrap.

  • flex-start: Lines are packed toward the top (in a row) or left (in a column).
  • flex-end: Lines are packed toward the bottom (in a row) or right (in a column).
  • center: Lines are centered.
  • space-between: Lines are evenly distributed, with space between them.
  • space-around: Lines are evenly distributed, with space around them.
  • stretch (default): Lines stretch to fill the container.

Example

                .container {
                    align-content: space-between; /* Distributes lines evenly */
                  }
