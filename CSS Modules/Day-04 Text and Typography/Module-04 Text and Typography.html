Text and Typography in css

Typography is the visual component of the written word. A text is a sequence of words.

1. Font Properties

1.1 font-family

The font-family property specifies the font to be used for text. It allows you to choose between system fonts, web-safe fonts, or custom fonts (like Google Fonts).


            /* Example: Setting font-family */
            body {
              font-family: 'Arial', sans-serif; /* Arial as primary, sans-serif as fallback */
            }

You can also load custom fonts using @font-face or external sources like Google Fonts:


                @import url('https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap');

                body {
                  font-family: 'Roboto', sans-serif;
                }

1.2 font-size

The font-size property controls the size of the text. It's essential to use relative units (like em, rem, or %) for responsive design.


            /* Example: Setting font-size */
            h1 {
              font-size: 36px;
            }
            
            p {
              font-size: 16px;
            }
            
            body {
              font-size: 1rem; /* 1rem equals the root font size */
            }

1.3 font-weight

The font-weight property defines the thickness of the text. Common values are normal, bold, or numerical values like 100, 400, 700 (depending on the font's available weights). 


                /* Example: Setting font-weight */
                h1 {
                  font-weight: bold;
                }
                
                p {
                  font-weight: 400; /* Normal weight */
                }

1.4 font-style

The font-style property allows you to specify italicized or oblique text. 


                /* Example: Setting font-style */
                em {
                  font-style: italic;
                }
                
                p {
                  font-style: normal; /* Default, no italics */
                }

1.4 font-style

The font-style property allows you to specify italicized or oblique text. 


                /* Example: Setting font-style */
                em {
                  font-style: italic;
                }
                
                p {
                  font-style: normal; /* Default, no italics */
                }

1.5 font-variant

This property is used for typographic features like small caps. 


                /* Example: Using font-variant */
                h1 {
                  font-variant: small-caps;
                }

2. Text Alignment and Spacing

2.1 text-align

The text-align property controls the horizontal alignment of text. Common values are left, right, center, and justify. 


                /* Example: Setting text-align */
                h1 {
                  text-align: center;
                }
                
                p {
                  text-align: justify;
                }

2.2 text-transform

The text-transform property changes the case of text. Common values are uppercase, lowercase, and capitalize. 


                /* Example: Using text-transform */
                h2 {
                  text-transform: uppercase; /* Converts text to uppercase */
                }

2.3 letter-spacing

The letter-spacing property controls the spacing between characters in the text. It's commonly used to adjust the appearance of headlines or large text. 


                /* Example: Using letter-spacing */
                h1 {
                  letter-spacing: 2px;
                }

2.4 line-height

The line-height property controls the amount of vertical space between lines of text. It improves readability, especially in blocks of text. 

/* Example: Setting line-height */
                p {
                  line-height: 1.6; /* 1.6 times the font size */
                }

2.5 text-indent

The text-indent property sets the indentation of the first line of a paragraph. 

/
                * Example: Using text-indent */
                p {
                  text-indent: 30px;
                }

3. Text Decoration and Effects

3.1 text-decoration

The text-decoration property controls text decoration, such as underlining, strikethrough, or overlining. 


                /* Example: Using text-decoration */
                a {
                  text-decoration: none; /* Removes underline from links */
                }
                
                h2 {
                  text-decoration: underline;
                }

3.2 text-shadow

The text-shadow property adds a shadow effect to the text. It accepts values for horizontal offset, vertical offset, blur radius, and color.


                /* Example: Adding text shadow */
                h1 {
                  text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.2);
                }

4. Responsive Typography

4.1 Using em and rem

  • rem is relative to the root font size (html element).
  • em is relative to the font size of the parent element. 

                /* Example: Using rem for responsiveness */
                html {
                  font-size: 16px; /* 1rem = 16px */
                }
                
                h1 {
                  font-size: 2rem; /* 32px */
                }
                
                p {
                  font-size: 1rem; /* 16px */
                }

4.2 Media Queries for Typography

Media queries help change font sizes and layout based on screen size or device. 


                /* Example: Responsive typography with media queries */
                h1 {
                  font-size: 3rem;
                }
                
                @media (max-width: 768px) {
                  h1 {
                    font-size: 2rem; /* Smaller font size on smaller screens */
                  }
                }

5. Advanced Typography Features

5.1 @font-face

Use @font-face to include custom fonts that aren't already available on a user's system. 


                /* Example: Using @font-face to load a custom font */
                @font-face {
                  font-family: 'MyCustomFont';
                  src: url('my-font.woff2') format('woff2'),
                       url('my-font.woff') format('woff');
                }
                
                body {
                  font-family: 'MyCustomFont', sans-serif;
                }

5.2 Web Fonts (e.g., Google Fonts)

You can use web fonts from external providers like Google Fonts to enhance typography. 


                /* Example: Importing Google Fonts */
                @import url('https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap');
                
                body {
                  font-family: 'Roboto', sans-serif;
                }

6. Text Overflow

The text-overflow property controls how text behaves when it overflows its container (often used with white-space: nowrap and overflow: hidden). 


                /* Example: Handling text overflow */
                p {
                  white-space: nowrap;
                  overflow: hidden;
                  text-overflow: ellipsis; /* Adds "..." when text overflows */
                }

7. CSS Variables for Typography

Using CSS variables makes it easier to maintain consistent typography throughout a website. 


                /* Example: Using CSS Variables for consistent typography */
                :root {
                  --main-font: 'Arial', sans-serif;
                  --font-size-base: 16px;
                  --line-height-base: 1.5;
                }
                
                body {
                  font-family: var(--main-font);
                  font-size: var(--font-size-base);
                  line-height: var(--line-height-base);
                }
