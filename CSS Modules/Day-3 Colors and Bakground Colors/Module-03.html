CSS Selectors

CSS selectors are used to "find" (or select) the HTML elements you want to style.

Introduction


In CSS (Cascading Style Sheets), selectors are used to target specific HTML elements to apply styles to them. A selector defines which elements the styles will be applied to based on various criteria such as element types, classes, IDs, attributes, or relationships between elements.

1. Universal Selector (*)

Purpose: Targets all elements in the document.

Syntax: 
* {
            property: value;
          }
          
Example: 
* {
            margin: 0;
            padding: 0;
          }

2. Type Selector (Element Selector)

Purpose: Targets elements of a specific type (e.g., p , div). 

Syntax: 

            element {
                property: value;
              }
              
          
Example: 

            p {
                font-size: 16px;
              }

3. Class Selector (.)

Purpose: Targets elements with a specific class.

Syntax: 

            .class-name {
            property: value;
          }
          
          
Example: 

            .button {
            background-color: blue;
          }

4. ID Selector (#)

Purpose: Targets an element with a specific ID (unique within the page). 

Syntax: 

            #id-name {
                property: value;
              }
              
          
Example: 

            #header {
                font-weight: bold;
              }

5. Descendant Selector (Space)

Purpose: Targets elements that are descendants of another element. 

Syntax: 

            ancestor descendant {
                property: value;
              }
              
          
Example: 

            div p {
                color: red;
              }

6. Child Selector (>)

Purpose: Targets direct children of an element. 

Syntax: 

            parent > child {
                property: value;
              }
              
          
Example: 

            ul > li {
                list-style-type: none;
              }

7. Adjacent Sibling Selector (+)

Purpose: Targets an element that is immediately preceded by a specific sibling element. 

Syntax: 

            element1 + element2 {
                property: value;
              }
              
          
Example: 

            h1 + p {
                margin-top: 0;
              }

8. General Sibling Selector (~)

Purpose: Targets any sibling element that follows the first element (not necessarily immediately). 

Syntax: 

            element1 ~ element2 {
                property: value;
              }
              
          
Example: 

            h1 ~ p {
                color: green;
              }

9. Attribute Selector

Purpose: Targets elements with a specific attribute or attribute value. 

Syntax: 

            element[attribute="value"] {
                property: value;
              }
              
          
Example: 

            a[href^="https"] {
                color: green;
              }

              input[type="text"] {
                border: 1px solid gray;
              }

10. Pseudo-classes (:)

Purpose: Targets elements in a specific state. 

Syntax: 

            element:pseudo-class {
                property: value;
              }
              
          
Example: 

            a:hover {
                color: red;
              }
              p:first-child {
                font-weight: bold;
              }

11. Pseudo-elements (::)

Purpose: Targets specific parts of an element, such as the first letter, first line, or inserting content before/after an element. 

Syntax: 

            element::pseudo-element {
                property: value;
              }
              
          
Example: 
p::first-letter {
            font-size: 2em;
          }

12. Group Selector (,)

Purpose: Combines multiple selectors to apply the same styles to different elements. 

Syntax: 

            selector1, selector2, selector3 {
                property: value;
              }
              
          
Example: 

            h1, h2, h3 {
                font-family: Arial, sans-serif;
              }

13. Combinator Selectors

Purpose: Combines different selectors to apply styles based on relationships between elements.

Examples:
            Descendant (space): div p selects all < p> inside < div>.
            Child (>): ul > li selects direct < li> children of < ul>.
            Adjacent sibling (+): h1 + p selects the first < p> after an < h1>.
            General sibling (~): h1 ~ p selects all < p> following an < h1>.

14. Negation Pseudo-class (:not())

Purpose: Targets elements that do not match a certain selector.

Syntax: 

            element:not(selector) {
            property: value;
          }
          
          
Example: 

            div:not(.active) {
            background-color: lightgray;
          }
