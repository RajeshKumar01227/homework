Module 13: Mini Project

  • Topics:

    • Plan and outline a simple webpage using learned HTML concepts

    • Design elements to include: text, images, links, and forms

  • Activities:

    • Work individually or in groups to build a simple webpage.

    • Present the project to the class for feedback.

  • Objectives:

    • Apply knowledge gained throughout the course to a practical project.

    • Collaborate and communicate effectively in project work.
