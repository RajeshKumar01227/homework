Display Properties in CSS

Display: Block Disply: Inline Display: Inline-Block Display: None

In CSS (Cascading Style Sheets), display properties define how an element should be displayed on the web page. Below are the main display property values:

1. Display: Block

  • Elements with display: block; take up the full width of their parent container, causing them to stack vertically.
  • They start on a new line, and any content following them appears below them.
  • example: < div>, < p>, < h1>, etc.
                div {
            display: block;
          }

2. Display: Inline

  • Elements with display: inline; do not start on a new line.
  • They only take up as much width as their content.
  • example: < span>, < a>, < strong>, etc.
                span {
                    display: inline;
                  }

3. Display: inline-block

  • This is a mix between block and inline.
  • It allows elements to be inline (side by side), but still have block-like properties (you can set width, height, margin, and padding).
  • example: < img>, custom styled divs.
                img {
                    display: inline-block;
                  }

4. Display: none

  • Elements with display: none; are removed from the document flow entirely and will not occupy any space.
  • They are not visible on the page, and they don't affect the layout of other elements.
  • This is different from visibility: hidden, which hides the element but still takes up space. 
                p {
                    display: none;
                  }
