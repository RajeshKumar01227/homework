Color Format in CSS

The color CSS property sets the foreground color value of an element's text and text decorations, and sets the currentcolor value.

1. Named Colors

PYou can use predefined color names (e.g., red, blue, green, etc.)

Example: 

            color: red;
            background-color: skyblue;

2. Hexadecimal (Hex) Colors

Hex color codes are a 6-digit representation of a color, with each pair of digits representing the red, green, and blue components in hexadecimal.

Example:

            color: #FF5733; /* Red */
            background-color: #008080; /* Teal */

3. RGB (Red, Green, Blue)

The RGB format allows you to define colors by specifying the intensity of red, green, and blue channels using values between 0 and 255.

Example:

            color: rgb(255, 87, 51); /* Red */
            background-color: rgb(0, 128, 128); /* Teal */

4. RGBA (Red, Green, Blue, Alpha)

RGBA is similar to RGB, but it also includes an alpha value to specify opacity, with 1 being fully transparent and 1 being fully opaque. 

Example:

            color: rgba(255, 87, 51, 0.7); /* 70% opacity red */
            background-color: rgba(0, 128, 128, 0.5); /* 50% opacity teal */

5. HSL (Hue, Saturation, Lightness)

HSL color model defines a color based on its hue, saturation, and lightness.
  • hue: angle on the color wheel (0 to 360 degrees)
  • saturation: percentage (0% to 100%)
  • lightness: percentage (0% to 100%)

Example:

            color: hsl(9, 100%, 60%); /* Red */
            background-color: hsl(180, 100%, 25%); /* Teal */

6. HSLA (Hue, Saturation, Lightness, Alpha)

Similar to HSL, but with an alpha value for transparency.

Example:

            color: hsla(9, 100%, 60%, 0.7); /* 70% opacity red */
            background-color: hsla(180, 100%, 25%, 0.5); /* 50% opacity teal */

7. CSS Color Functions

  • color: currentColor;: Uses the current value of the color property.
  • color: transparent;: Fully transparent color.

8. Opacity

You can also use the opacity property to set the transparency of an element.

Example:

            element {
            opacity: 0.5; /* 50% opacity */
          }
